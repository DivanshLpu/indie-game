# How to use releases

1. Open the **Releases** page for this project.
2. Open the latest release.
3. Read the release notes.
4. Download the **ZIP** file from the release notes.
5. Install **LÖVE / Lua support** for your device if it is not already installed.

## On PC or laptop

- Download the ZIP from the release notes.
- Install LÖVE for your operating system.
- Drag and drop the downloaded ZIP onto `love.exe` or the `love.exe` shortcut.
- The game will start.

## On mobile

- Open the **LÖVE launcher** app.
- Browse to the downloaded ZIP from the release notes.
- Open the ZIP in the launcher.
- The game will start.

## If you download from release assets and get extracted files

Sometimes the release asset may contain extracted game files instead of a ready-to-run ZIP.

1. Extract everything.
2. Open the extracted **indie game** folder.
3. Go inside the inner folder that contains the actual game files.
4. Select **all files and folders** inside that inner folder.
5. Create a new ZIP from those selected files.
6. Open the newly created ZIP with the **LÖVE launcher** app or drag it onto `love.exe` on PC.

## Result

After that, the game should run.
