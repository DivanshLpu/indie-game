# 📖 About

This is my personal **hobby game development project** built with **Lua** and **LÖVE2D**.

The goal is **not** to recreate Pokémon, but to capture the feeling of classic **Game Boy Advance RPGs**—pixel art, top-down exploration, smooth gameplay, and retro aesthetics—while building everything from scratch using modern open-source tools.

The project currently targets **PC**, with plans to support **Android**, **iOS**, and other platforms in the future.

# 🚀 Getting Started

## 1️⃣ Install LÖVE2D

Download and install the latest version of **LÖVE2D** for your operating system.

---

## 2️⃣ Create a Shortcut

After installing LÖVE2D:

* Locate the **love.exe** executable.
* Create a shortcut.
* Copy the shortcut into your project's root directory.

---

## 3️⃣ Create the Project Folder

Inside the root directory, create a folder named:

```text
indie-game
```

Or simply clone this repository into that folder.

---

## 4️⃣ Develop the Game

Write all your game code inside the **indie-game** folder.

Example structure:

```text
Root/
│
├── Love2D Shortcut
│
└── indie-game/
    │
    ├── main.lua
    ├── conf.lua
    │
    ├── assets/
    │   ├── sprites/
    │   ├── tilesets/
    │   ├── maps/
    │   ├── audio/
    │   └── fonts/
    │
    ├── src/
    │
    ├── libraries/
    │
    └── save/
```

---

## 5️⃣ Run the Game

After development:

* Return to the root directory.
* Drag the **indie-game** folder onto the **Love2D shortcut**.
* The game will automatically launch.
