About The Game

A Side Project Born From Passion

This indie RPG game is a personal hobby project that I've been developing in my free time. As someone who has always been fascinated by classic role-playing games, I wanted to create something that captures the essence of old-school RPGs while adding modern touches.

Built with LÖVE2D and Lua, this game features turn-based combat, exploration, character progression, and a story-driven experience. Every aspect of the game—from the pixel art to the game mechanics—has been crafted with care and attention to detail.

This is an ongoing project that I continue to develop and improve. Your feedback is incredibly valuable in helping shape the future of this game!
